Blocks

In ruby the most basic definition of a block is the code between curly braces { } or do and end.

In almost all cases you can use either one and they mean the same thing. To keep things simple let’s ignore the differences.

I used to have a hard time understanding ruby procs and lambdas until I started to write out what they represented.
